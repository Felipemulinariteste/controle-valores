Controle de Valores Guardados

Anote os valores guardados e depositados e veja o total em dinheiro atualizado automaticamente.

O app funciona sem Internet depois do primeiro carregamento.

Como instalar no celular

Abra o link no Chrome, navegue alguns segundos dentro do app e depois use o menu do navegador se o botão de instalação não aparecer.

  1. Abra o link HTTPS do GitHub Pages no Chrome.
  2. Toque no menu do navegador.
  3. Escolha Instalar app ou Adicionar à tela inicial.
