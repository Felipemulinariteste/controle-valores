# Controle de Valores Guardados - GitHub Pages

## Como publicar
1. Crie um repositório no GitHub.
2. Envie estes arquivos para a **raiz** do repositório.
3. Vá em **Settings > Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Selecione a branch **main** e a pasta **/ (root)**.
6. Salve.
7. Aguarde o link HTTPS do GitHub Pages aparecer.
8. Abra esse link no celular e instale o app.

## Observação
Depois de publicado, o app fica disponível gratuitamente no GitHub Pages e com HTTPS automático.
